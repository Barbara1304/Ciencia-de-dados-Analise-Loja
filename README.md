# Análise de Lojas – Relatório para Decisão Estratégica

Este projeto foi desenvolvido para auxiliar o Sr. João na decisão sobre qual das quatro lojas vender, utilizando dados reais e indicadores consolidados de desempenho.

A análise foi feita em Python, utilizando Pandas para manipulação de dados e Matplotlib para gerar visualizações.

---

## Objetivo do Projeto

Avaliar as quatro lojas e identificar aquela que apresenta o pior desempenho geral.  
Os dados apontam que a Loja 4 é a mais indicada para ser vendida.

---

## Stack Utilizada

- Python 3  
- Pandas  
- Matplotlib  
- Google Colab / VS Code  
- Git e GitHub

---

## Funcionalidades da Análise

- Comparação de faturamento por loja  
- Análise da avaliação média dos clientes  
- Comparação do frete médio  
- Identificação da loja com pior desempenho geral  
- Visualização dos resultados em gráficos

---

## Como Rodar o Projeto

1. Baixe o arquivo `.ipynb` do repositório  
2. Abra no Google Colab ou no VS Code  
3. Execute as células em ordem  
4. Os gráficos e resultados aparecerão automaticamente
